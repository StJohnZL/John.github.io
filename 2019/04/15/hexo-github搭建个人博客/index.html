Hexo+Github搭建个人博客

Posted on 2019-04-15 | Edited on 2019-04-23

一、组成部件

1. Hexo

​ Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

2.GitHub pages

​ 使用github pages来托管Hexo生成的静态网页。

三、Hexo命令

  1. hexo g：

    在现有文件下生成静态网页，每次修改或者写博客之后都要先生成静态网页。

  2. hexo s：

    本地展示静态页面。

  3. hexo d：

    部署静态页面到远程地址，比如：GitHub。

  4. hexo clean：

    清空。

  5. hexo new ‘blog-name’

    在source/_post/下创建md文件，在其中可以写相应内容，之后hexo g生成静态页面，最后用hexo d部署到远程服务端。
